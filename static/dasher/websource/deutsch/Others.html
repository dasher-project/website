Eine "Zirkular"-Version von Dasher in Java  In Anbetracht der Anerkennung unserer Arbeit meinen wir, dass die hier genannte Implementierung die essentielle Besonderheit von Dasher aufgibt, nämlich die Informationseffizienz. Bei der Javaversion kann sich der Anwender in Zwischenräume verirren, die mit keiner Zeichenfolge korrespondieren. Die Bandbreite von Dasher wird dadurch verdorben, dass der Anwender mit einer gewissen Redundanz in die codierenden Regionen steuern muss.
