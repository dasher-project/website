Outras interfaces similares ao Dasher, escritas fora de nossa grupo

Uma versão "circular" do Dasher, escrita em Java Apesar de apreciarmos a homenagem a nosso trabalho, achamos que esta implementação desperdiça o elemento essencial do Dasher, a saber, a eficácia de informação. O usuário é capaz de fazer o zoom em regiões intercirculares que não correspondem a nenhuma seqüência de caracteres e, logo, desperdiça-se largura de banda continuamente, pois o usuário tem que redirecionar (de maneira redundante) para as regiões de codificação.
