 Este é o display do Dasher. Você está prestes a assistir o Dasher
sendo usado para escrever "hello how are you".

À direita, estão as letras do alfabeto, em ordem, com 
um traço de sublinhado no final, que representa um espaço.

À medida que a seta do mouse se move para a direita, as letras rolam pela
tela.

Cada letra se encontra em uma caixa. Em cada caixa se encontra um outro alfabeto completo e,
dentro de cada letra desses alfabetos, há um outro alfabeto, e assim por diante.

O segredo do Dasher é que o tamanho das caixas de letra depende
da probabilidade com que cada letra pode ser escolhida na seqüência, tendo por base o que já
foi escrito. Letras improváveis são, com freqüência, pequenas demais para aparecer, motivo
por que os alfabetos podem parecer imcompletos. Letras
prováveis tem caixa grandes, de modo que é fácil e rápido selecionar as próximas
letras, na maioria das vezese. Assim que você tiver começado a escrever a primeira ou 
ou as duas primeiras letras de uma palavra ou expressão de uso comum, o software,
com freqüência, pode predizer até mesmo a próxima palavra inteira, o que torna ainda
mais fácil e rápido escrever.

Quanto mais para a direita a seta estiver, mais rápido as letras se
moverão. Para reduzir a velocidade, mova-a para a esquerda. Para selecionar letras,
mova o ponteiro para cima ou para baixo. É como pilotar. Se não estiver vendo
a letra que deseja, basta direcionar o ponteiro aonde ela estaria
na ordem alfabética. Ela aparecerá num instante.

É divertido! E lembre-se: esta demonstração teve a velocidade reduzida para
facilitar a percepção de como o Dasher funciona. Com um pouco de prática, você poderá
escrever muito mais rápido.
