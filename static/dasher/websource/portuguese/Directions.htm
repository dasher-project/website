Rumos futuros do projeto Dasher

O projeto Dasher está sendo levado adiante pelo Inference group no laboratório Cavendish; outros pesquisadores que compartilham o desejo de ver o Dasher largamente utilizado no mundo real também estão convidados a trabalhar no Dasher.

O foco inicial é o desenvolvimento do Dasher para usuários incapacitados; são metas adicionais o desenvolvimento da versão para palmtop; o desenvolvimento das versões em japonês; e a criação de versões especializadas para computadores específicos. Para levarmos todas essas metas adiante, estamos liberando o código-fonte do Dasher como um projeto de Código Aberto (data estimada, setembro de 2002).

Sonhos para o futuro

Sistema híbrido voz-dasher
Fale em um reconhecedor discursivo imperfeito e veja suas inferências exibidas segundo a previsibilidade; sempre que não tiver certeza sobre o que foi dito, o sistema pode usar o dasher para seguir para a sentença correta. Muito mais fácil do que ter que corrigir erros dizendo comandos discursivos adicionais!
Sistema híbrido tradutor automático-dasher
Suponha que tenhamos um sistema de tradução deficiente que traduz muito mal do francês para o inglês. Um especialista precisa conferir a tradução e arrumar os erros. Esse serviço poderia ser feito no Dasher, usando a saída do tradutor para definição do modelo de linguagem.
