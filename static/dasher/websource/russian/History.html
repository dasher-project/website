История Dasher

Кто есть кто

  • Dasher появился благодаря Inference Group, которую возглавляет Дэвид Маккэй, (David MacKay) лектор Департамента Физики и соучредитель IT-компании Transversal. Дэвид создал первый прототип Dasher'a в 1997-м году.
  • Дэвид Вард (David Ward) разрабатывал и улучшал версии Dasher'a c 1998-го года до 2002-го; для своей кандидатской диссертации Дэвид облёк концепцию Dasher в программную оболочку, постоянно совершенствовал систему, поставил множество экспериментов, показывающих качество работы системы. Сейчас Дэвид работает на Spiral Software в Кембридже.
  • Алан Блэкуэл (Alan Blackwell), лектор Компьютерной Лаборатории, помогал нам в постановке экспериментов.
  • Фил Коуэнс (Phil Cowans) и Тим Хоспедалес (Tim Hospedales) также внесли свой вклад, особенно в работах с системами слежения за движением глаз (eyetracking).
  • В течение лета 2002 года будет подготовлен к публикации, как the Open Source исходный код программы; готовит публикацию Айан Мёррэй (Iain Murray). Айан собирается писать свою кандидатскую диссертацию по вычислительной нейрологии в UCL.
  • Начиная с лета 2002 года, мы надеемся, к проекту Dasher на добровольной основе примкнут эксперты по программному обеспечению, что позволит ещё улучшить систему и адаптировать её к другим программно-аппаратным платформам.
  • Мы будем стараться максимально поддерживать менеджеров проектов в управлении этим "Open Source" проектом, чтобы успех был гарантированным.

Версии Dasher'a

tcl - Оригинальный прототип
Показывает, как выглядит исходный код; включает также "сырые" исходники для языковой модели.
написано Дэвидом Мккэем (David MacKay).
C и tcl - для Linux и Windows
Использует ppm в качестве языковой модели. Управляется мышью.
написано Дэвидом Вардом (David Ward).
C - для карманных PC
Управление - тачскрин.
написано Дэвидом Вардом (David Ward).
Eyetracker Dasher
Управляется мышью и устройством слежения за движением глаз (eyetracker).
написано Дэвидом Вардом (David Ward).
