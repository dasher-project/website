Versions de Dasher

Les principales versions fonctionnant de Dasher sont les suivantes:
Version 1.*.* - C et tcl - pour linux et windows desktops
Utilises ppm comme language modèle. Piloté par souris.
écrit par David Ward.
Version 1.*.* supporte une grande partie des langues Européenne et le Japonais (Hiragana). La version Anglaise peut supporter majuscules et minuscules. Le language modèle de cette version peut être instruite aussi bien par le chargement d'un fichier d'entrée exemple que par le chargement d'un dictionaire d'orthographe correcte.
C - pour les PC de poche
Pilotés par un stylet sur un écran tactile.
écrit par David Ward.
Cette version inclus majuscules, chiffres , et un certain nombre de ponctuations. Seulement pour l'Anglais.
Version 2.*.* - C - pour linux et windows desktops
Version 2.*.* pour l'Anglais, majuscule et minuscule, ponctuation et chiffres. Il sera réalisé en "Open Source" fin de l'été 2002.
Dasher oculaire
Piloté par souris contrôlée par un pointeur oculaire.
écrit par David Ward.
JDasher
Version langue Japonaise de Dasher (Hiragana) - incluse dans la version C et tcl
tcl - Prototype original

Demontres la relation avec un codage arithmétique; inclus un modèle brut de language bigram.
écrit par David MacKay.
Tourne sur toutes les plateformes qui supportent tcl (linux, windows).
