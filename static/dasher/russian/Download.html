Скачать Dasher

Программа Dasher (версии, начиная с 3-й и выше) распространяются на условиях, предлагаемых GNU-лицензией. Вы должны ознакомиться с условиями этой лицензии.

Чтобы скачать Dasher, пожалуйста, заполните форму на этой странице. Это необязательно, но мы были бы Вам очень благодарны за Вашу информацию о себе. Эта информация будет использована только для улучшения Dasher, и никто посторонний не получит доступ к этой информации. Мы также не будем заниматься рассылками материалов по предоставленным адресам. Если после загрузки программы Вы захотите связаться с разработчиками Dasher'a, то кликните эту ссылку, и Вы попадёте на страницу отправки сообщения разработчикам.

Microsoft Windows

  version 4.11 (Windows XP, Vista, Windows 7)
  version 4.6.2 (Windows 2000)
  32 bit Windows XP version is confirmed to work under 64 bit Windows 7.
  Please manually uninstall any versions prior to 4.4.2 before installing version 4.6.0.
